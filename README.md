# florVoiceGraph
> Simplifiying your voice box!

florVoiceGraph is a simplistic ui element that spices up your server by changing little things to make it unique!

## Installation

Easy!

```sh
Navigate to: yourservername > addons > Drag & Drop
```
and done!
